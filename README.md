# 2주 80시간 근무 계산기

정적 HTML 기반 근무시간 계산기입니다.  
출근/퇴근/휴게시간을 입력하면 일일 근무시간, 일일 초과/미달, 꿀근무 시간, 주 40시간/2주 80시간 달성 현황을 계산합니다.

## 주요 기능

- 일별 입력: 출근, 퇴근, 휴게(분), 휴일 여부, 휴가(연차/반차/반반차)
- 자동 계산:
  - `일일 근무` (휴게 차감 + 체류구간 의무휴게 반영)
  - `일일 초과/미달` (일일 근무 - 8시간)
  - `꿀근무` (출근 + 8시간 29분)
  - 최소 4시간 충족 여부
- 집계:
  - 1주차 총 시간, 2주차 총 시간, 2주 누적 총 시간
  - 1주차/2주차 마지막 근무일 목표 퇴근시간
  - 전체 초과/미달 합계
- 대량 입력:
  - 탭/줄바꿈 형식 데이터 붙여넣기 후 일괄 반영
- 예시 데이터 자동 입력 버튼 제공

## 입력 형식

### 표 직접 입력

- 출근/퇴근: `HH:MM:SS` (24시간)
- 휴게: 분 단위 숫자 (예: `0`, `30`, `60`)
- 휴일 체크 시 해당 일자는 0시간 처리
- 휴가:
  - 없음: 0시간
  - 반반차: 2시간
  - 반차: 4시간
  - 연차: 8시간

### 붙여넣기 입력

다음 형태를 텍스트 박스에 붙여넣고 `붙여넣기 적용` 버튼 클릭:

```text
08:15:48	20:06:19		30
08:04:31	22:01:43		0
08:01:24	16:08:06		0
07:47:43	18:00:15		30
08:12:24	16:52:12		0
```

- 1열: 출근
- 2열: 퇴근
- 마지막 숫자열: 휴게(분)
- 빈 줄은 휴일로 해석

## 계산 규칙

### 1) 일일 근무

`일일 근무 = 체류시간 - 입력 휴게 - 의무 휴게`

- 체류시간: `퇴근 - 출근` (자정 넘어가면 다음날로 처리)
- 의무 휴게(체류시간 기준):
  - 4시간 미만: 0분
  - 4시간 이상 ~ 8시간 30분 미만: 30분
  - 8시간 30분 이상 ~ 12시간 미만: 60분
  - 12시간 이상: 90분

### 2) 일일 초과/미달

`일일 초과/미달 = 일일 근무 - 8시간`

- 양수: 초과
- 음수: 미달

### 3) 꿀근무

`꿀근무 = 출근 + 8시간 29분`

### 4) 주/2주 집계

- 1주 기준: 40시간
- 2주 기준: 80시간
- 휴가시간(연차/반차/반반차)은 총 시간 계산에 포함
